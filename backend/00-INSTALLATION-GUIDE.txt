## INSTALLATION STEPS:

### 1. CREATE PROJECT DIRECTORY
```bash
mkdir BBMS-Backend
cd BBMS-Backend
```

### 2. CREATE SUBDIRECTORIES
```bash
mkdir models controllers routes middleware
```

### 3. COPY FILES TO APPROPRIATE FOLDERS
- Copy server.js, package.json, .env, .gitignore to root
- Copy all model files to models/ folder
- Copy all controller files to controllers/ folder
- Copy all route files (auth.js, donors.js, etc.) to routes/ folder
- Copy middleware/auth.js to middleware/authMiddleware.js

### 4. INSTALL DEPENDENCIES
```bash
npm install
```

### 5. UPDATE .env FILE
Edit .env and change JWT_SECRET to a strong random key

### 6. START SERVER
```bash
npm run dev
```
